Past History

The website started as my 20 years-old birthday (self-)gift, and iterates a new version every year.

Version 3 (2024, Current, 22 years old)

Utilizing HTML5 UP and a more professional mindset, I aimed to render this version as both a personal portfolio and a resource hub.

  • Included separate html files covering five aspects.
  • Featured more photos and website functions for interactiveness.
  • Implemented backend functionalities like mailing feedback systems and mentee-exclusive access.

Version 2 (2022-23, 21 years old)

Inspired by a Github open-sourced project, this version focused on personal branding and carving a space to share insights.

  • This version was my first attempt at working with CSS to standardize containers and layout elements. One html file contains everything.
  • Featured more homemade content, storing most necessary information in two Google Drive accounts.
  • Invited a few juniors to co-develop a QFIN website.

  • View Version 2

Version 1 (2022, 20 years old)

Based on one HKUST CSE PA template, this version reflected my early curiosity in web development, and to represent academic interests and projects.

  • It mostly served as an aggregation of information pointers, containing "Dashboard" and "About".
  • Content-wise, attempted to feature academic information, clubs intro and new student information.
  • Being a (very apprentice) webpage developer sounded fun!
